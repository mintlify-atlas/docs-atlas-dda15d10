    <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
      Get ReplaySorcery up and running in minutes
    </p>
  </div>

  <Steps>
    <Step title="Install ReplaySorcery">
      On Arch Linux, install from the AUR:
      
      ```bash
      yay -S replay-sorcery
      ```
      
      For other distributions, [build from source](/installation#building-from-source).
    </Step>

    <Step title="Enable the systemd service">
      Start ReplaySorcery as a user service to run in the background:
      
      ```bash
      systemctl --user enable --now replay-sorcery
      ```
      
      The service will continuously record your screen in a circular buffer.
    </Step>

    <Step title="Save your first replay">
      Press **Ctrl+Super+R** to save the last 30 seconds of gameplay. Your video will be saved to `~/Videos/ReplaySorcery/`.
      
      <Note>
        The default keybinding can be customized in `~/.config/replay-sorcery.conf`
      </Note>
    </Step>

    <Step title="Configure video settings (optional)">
      Edit `~/.config/replay-sorcery.conf` to customize recording parameters:
      
      ```bash
      # Change recording duration
      recordSeconds = 60
      
      # Enable hardware acceleration
      videoInput = hwaccel
      
      # Set video quality (lower is better)
      videoQuality = 23
      ```
      
      Restart the service after making changes:
      ```bash
      systemctl --user restart replay-sorcery
      ```
    </Step>
  </Steps>
</div>
